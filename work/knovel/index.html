Knovel

Engineering a Platform

Knovel is a web application designed to surface technical information engineers need to solve problems. We redesigned the experience from the ground up.

Forging an Experience

PDF data was an integral part of Knovel’s library so we built a brand new reader. The search interface reveals important data without fuss. The breadcrumbs and web url change dynamically while browsing to preserve the user’s page. And full screen reading allows for a distraction-free experience.
